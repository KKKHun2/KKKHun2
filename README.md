<h2>안녕하세요! 신입 프론트엔드 개발자 김용훈입니다.</h2> <br />
JavaScript와 TypeScript를 기반으로 다양한 React 및 Next.js 프로젝트들을 개발해왔습니다. <br />
CSS, SCSS, Styled-Components, TailwindCSS 등 여러 스타일링 방식에 능숙하며, 프로젝트 성격에 맞는 UI 구조를 설계하고 구현할 수 있습니다.<br />
<br />
진행한 프로젝트로 LG Enact 기반의 TV/WebOS 개발 경험과 기존 JSP 기반의 서비스를 React로 마이그레이션한 경험을 보유하고 있으며,<br />
이를 통해 레거시 시스템 개선과 모던 프런트엔드 구축 모두에 강점을 갖고 있습니다.<br />
<br />
또한 소규모 진행이긴 했으나 대기업 앱 프로젝트를 프론트 개발 리더로서 주도한 경험이 있습니다.<br />
마이그레이션 과정에서는 단순 이전에 그치지 않고, 서비스 구조 재정비와 신규 기능 추가를 함께 진행하여 사용자 경험을 강화했습니다.<br />
프로젝트 진행 중 네이티브 개발자분들과의 긴밀한 협업을 통해 웹과 네이티브 간 인터페이스 정의·기능 연동·이슈 조율을 이끌어내며 안정적인 앱 서비스 구조를 완성했습니다.<br />
<br />
API 호출 구조는 주로 React Query를 활용하여 구성했으며, Redux 역시 필요한 전역 상태 관리에 사용했습니다.<br />
React Query의 캐싱 전략과 staleTime 관리로 데이터 호출 빈도를 줄이고, 불필요한 네트워크 요청을 최소화하며,
서버 상태 기반의 데이터 동기화 및 성능 최적화 작업을 진행했습니다.<br />
<br />
새로운 기술을 빠르게 습득하고 업무에 효과적으로 적용하는 데 자신 있으며, 문제 상황에서도 구조적이고 유연한 접근으로 해결 방안을 마련합니다.<br />
협업 시에는 팀 간 커뮤니케이션을 중요시하며, 의견 조율과 기능 개발을 함께 진행하며 높은 완성도의 결과물을 만들어 왔습니다.<br />
